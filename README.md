# AI Markating Agency Website

Premium futuristic React website for `www.ai-markatingagency.com/`.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The contact form opens an email draft to `Mohitdhiman8383@gmail.com` and includes a WhatsApp CTA for `+91 78143 23946`.

## GitHub Pages

This project includes a GitHub Actions workflow at `.github/workflows/deploy.yml`.
After pushing to a public GitHub repository, enable Pages with the source set to
GitHub Actions. The live site will be published by the workflow.
